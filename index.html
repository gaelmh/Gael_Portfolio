Projects

NBA Free Throw Accuracy

Through data visualizations, this project investigates NBA player statistics from the 1997-98 to the 2021-22 seasons to analyze factors affecting free throw accuracy and its correlation with MVP status in the NBA.

Technologies: R (tidyverse, GGally, corrplot, magrittr, ggforce, reshapeggplot)

Data Source: The data was taken from the "NBA Player Stats (1998-2022)" dataset from Kaggle.

Techniques: Data Filtering, and Visualizations.

Understanding Healthcare Costs – A Generalized Approach

Through generalized linear models, this project investigates healthcare data to analyze the impact of demographic and lifestyle factors on healthcare costs.

Technologies: R (tidyverse, caret, lmtest, broom, car, MASS, fastDummies)

Data Source: The data was taken from the “Medical Costs Personal” dataset from Kaggle.

Techniques: Linear Regression, Logistic Regression, and Analysis of Variance (ANOVA).

Boot Brand Preference Among Soccer Player – A Machine Learning Approach

Leveraging machine learning modeling, this project investigates soccer boots data to predict the preferred boot brand of a given soccer player, and identify the key factors influencing their preference.

Technologies: Python (pandas, numpy, collections, scikit-learn, joblib, imblearn, sys, math, urllib, os, matplotlib, seaborn)

Data Source: The data was taken from the “Football Boots” dataset from Kaggle.

Techniques: Class Balancing, Feature Selection, and Machine Learning Modeling.

Soccer Starting XI Prediction

Leveraging AWS tools and machine learning models, this project investigates soccer match data to predict whether a soccer player will be in the starting lineup for an upcoming game.

Technologies: Python (pandas, numPy, scikit-learn, imblearn, matplotlib, seaborn, boto3), Apache Spark (PySpark), AWS (S3, Athena)

Data Source: The data was taken from the “Football Data from TransferMarkt” dataset from Kaggle.

Techniques: AWS tools (EMR, S3, and Athena), Class Balancing, Data Staging and Querying, and Modeling.

Predicting Healthcare Costs – A Machine Learning Approach
